Небольшое пояснение по работе сервиса: по заданию необходимо передавать входные данные в теле запроса в формате JSON. Наш проект работает с данными ЭКГ, которые в любом случае требуют предварительной обработки специальной библиотекой. Для удобства работы с сервисом было принято решение реализовать прямой импорт файлов ЭКГ. 

Необходимые для тестирования работы сервиса файлы находятся в папке /**files_for_testing**. Для работы сервиса требуется передать данные пациента в *header*, либо загрузить *csv*-файл с метаданными. Также загружаются два файла: *.dat* и *.hea*. 
**Важно**: оба файла дублируются исходя из разрешения записи - *lr* или *hr*. *hr* - высокое разрешение и предпочтительно использовать его, т.к. качество записи выше. Но модель может работать и с *lr* - низким качеством записи.
